# PRG1 — Programación I

- **Nivel**: Nivel 0
- **Área de conocimiento**: Fundamentos de programación y software
- **Fuente canónica**: MIT 6.100L + Harvard CS50P + Python Tutorial (es) + Think Python 3e

## Objetivo de la materia

Escribir, ejecutar y depurar programas en Python que resuelvan problemas concretos usando variables, control de flujo, estructuras de datos y funciones.

## Temas centrales y objetivos de aprendizaje

### 1. Fundamentos del lenguaje

**Conceptos que lo integran**: qué es programar, el intérprete, print / I-O, variables, tipos (int, float, str, bool), type().

**Objetivos de aprendizaje** (nivel Bloom entre paréntesis):
- Explicar (Entender) qué es un intérprete y en qué orden ejecuta un programa.
- Predecir (Aplicar) el tipo que infiere Python para un valor dado.
- Escribir (Aplicar) programas que muestren y combinen valores de distintos tipos.

### 2. Strings y entrada

**Conceptos que lo integran**: strings, concatenación, f-strings, inmutabilidad, input(), conversión de tipos.

**Objetivos de aprendizaje** (nivel Bloom entre paréntesis):
- Construir (Aplicar) texto dinámico con f-strings.
- Justificar (Entender) por qué input() devuelve siempre str y cuándo hay que convertir.
- Detectar (Analizar) errores de conversión de tipos.

### 3. Control de flujo

**Conceptos que lo integran**: if / elif / else, operadores de comparación y lógicos, while, for, range, break, continue, loops anidados.

**Objetivos de aprendizaje** (nivel Bloom entre paréntesis):
- Elegir (Analizar) entre while y for según el problema.
- Escribir (Aplicar) condicionales con ramas mutuamente excluyentes y orden correcto.
- Trazar (Analizar) la ejecución de loops con break/continue.

### 4. Colecciones

**Conceptos que lo integran**: listas, métodos de listas, slicing, aliasing/clonado, diccionarios, tuplas, sets.

**Objetivos de aprendizaje** (nivel Bloom entre paréntesis):
- Seleccionar (Analizar) la estructura de datos adecuada (lista/dict/tupla/set) para un caso.
- Manipular (Aplicar) listas y diccionarios (acceso, actualización, recorrido).
- Explicar (Entender) el efecto del aliasing al copiar listas.

### 5. Funciones y errores

**Conceptos que lo integran**: def / return, parámetros y locales, parámetros por defecto, retorno múltiple, try/except.

**Objetivos de aprendizaje** (nivel Bloom entre paréntesis):
- Descomponer (Crear) un problema en funciones pequeñas y reutilizables.
- Escribir (Aplicar) funciones con parámetros por defecto y retorno múltiple.
- Manejar (Aplicar) errores esperables con try/except específico.

### 6. Proyecto integrador

**Conceptos que lo integran**: combinación de todo lo anterior, descomposición, pruebas por función.

**Objetivos de aprendizaje** (nivel Bloom entre paréntesis):
- Diseñar y construir (Crear) una solución completa a un caso realista con el nivel de un problem set de MIT 6.100L.
- Descomponer (Analizar) el problema en funciones independientes y probables por separado.
- Evaluar (Evaluar) la propia solución contra los tests y corregir los casos que fallan.
